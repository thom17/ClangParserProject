USAGE EXAMPLES
================================================================================

BEFORE:
-------
manager = ParseManager('/tmp/database.db')
manager.smart_parse_project('/path/to/project')

AFTER:
------
manager = ParseManager('/path/to/project')
manager.smart_parse_project()  # project_dir not needed

WITH FILTERS:
-------------
manager = ParseManager('/path/to/project')
manager.add_exclude_pattern('**/test/**')
manager.add_include_pattern('**/*.cpp')
results = manager.smart_parse_project(use_filters=True)

WITH CONFIG FILE:
-----------------
# Create .clangparse_ignore in project root
manager = ParseManager('/path/to/project')
manager.load_filter_config()  # Auto-loads .clangparse_ignore
results = manager.smart_parse_project()
